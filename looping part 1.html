Reverse

2WAP to accept any number and reverse it and display the new number. Example: Input Number - 123 Reverse number - 321

Reverse

Palindrome

3 WAP to accept any number and check the number is palindrome or not. Example: Input Number - 123 Reverse number - 321 Input value & reverse value both are not same, the 123 is not a palindrome number. Example: Input Number - 1221 Reverse number - 1221 Input value & reverse value both are same; the 1221 is a palindrome number.

Palindrome

Armstrong

4 WAP to accept any number and check the number is Armstrong or not. Example: Input Number - 153 C= 13+53+33 C=1+125+27= 153 Input value & calculated value both are same, the 153 is a Armstrong number.
